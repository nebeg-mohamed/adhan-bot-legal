# Adhan Bot Privacy Policy

**Effective Date:** 06/02/2026
**Bot Name:** Adhan
**Contact:** nebegmohamedinf@gmail.com

This Privacy Policy explains what information Adhan Bot collects, how it is used, and how it is stored.

## 1. Information We Collect

Adhan Bot only collects information needed to operate prayer reminders and server configuration.

The bot may store the following server-related data:

* Discord server ID
* Discord server name
* Selected text channel ID
* Selected text channel name
* Selected voice channel ID
* Selected voice channel name
* Selected country
* Selected city
* Reminder minutes
* Whether the bot is enabled for the server

## 2. Information We Do Not Collect

Adhan Bot does not intentionally collect:

* Private messages
* Message content from users
* User passwords
* Payment information
* Personal addresses
* Phone numbers
* Email addresses from Discord users
* Voice recordings
* Audio from voice channels

The bot only plays audio files and does not record or store voice channel audio.

## 3. How We Use the Data

The collected data is used to:

* Remember each server’s selected text channel.
* Remember each server’s selected voice channel.
* Remember the selected country and city for prayer times.
* Send prayer reminder messages.
* Play reminder audio and Adhan audio in the configured voice channel.
* Display current settings using `/settings`.

## 4. Data Storage

Server settings are stored in a database, such as MongoDB.

We take reasonable steps to protect stored data, but no online service can guarantee complete security.

## 5. Data Sharing

We do not sell server data or share it with advertisers.

Data may be processed through third-party services required for the bot to function, such as:

* Discord API
* Prayer time API providers
* Database hosting providers

These services may process data according to their own policies.

## 6. Data Retention

Server configuration data is stored as long as the bot is installed or until removal is requested.

If a server administrator wants the bot’s stored configuration deleted, they may contact us using the contact information below.

## 7. Data Removal

A server administrator may request deletion of stored server settings by contacting:

nebegmohamedinf@gmail.com

Please include the Discord server ID if possible so we can identify the correct data.

## 8. Children’s Privacy

Adhan Bot is intended for use within Discord servers and follows Discord’s platform rules. The bot does not knowingly collect personal information from children.

## 9. Changes to This Policy

We may update this Privacy Policy when the bot changes or when required by platform or legal requirements.

Continued use of the bot after updates means you accept the updated policy.

## 10. Contact

For questions, privacy requests, or data deletion requests, contact us at:
