## 1. Description of the Service

Adhan Bot is a Discord bot designed to help server administrators send prayer time reminders, play a 10-minute prayer reminder audio, and play the Adhan in a selected voice channel at prayer time.

The bot allows server administrators to configure:

* A text channel for prayer messages.
* A voice channel for Adhan and reminder audio.
* A country and city for prayer time calculations.
* Prayer reminder settings.

## 2. Server Administrator Responsibility

Server administrators are responsible for configuring the bot correctly using commands such as:

* `/settext`
* `/setvoice`
* `/setlocation`
* `/settings`

Administrators must ensure that the bot has the required permissions in the selected channels.

## 3. Required Discord Permissions

For the bot to work correctly, it may require:

In text channels:

* View Channel
* Send Messages
* Read Message History
* Manage Messages

In voice channels:

* View Channel
* Connect
* Speak
* Use Voice Activity

The Manage Messages permission is used to delete previous reminder messages when the prayer time arrives.

## 6. Prayer Time Accuracy

Prayer times are calculated using third-party prayer time APIs and depend on the selected country and city.

Different calculation methods, local authorities, or mosques may use different prayer times. Users should verify prayer times with local trusted sources when accuracy is important.
